photosets.css

to everyone who balks at having to use javascript and pxu to style photosets, we see you

photosets.css is a CSS stylesheet (compiled by JavaScript, ironically) that lays out all photoset layouts using solely CSS and a nifty photosets variable {PhotosetLayout}. The pictures are arranged using the CSS grid layout, though a fallback using inline blocks has been provided, in case anyone is using a browser that doesnâ€™t support CSS grid.
